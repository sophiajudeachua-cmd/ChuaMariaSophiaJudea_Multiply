# Description
This project is a basic Python utility that multiplies two user-provided numbers. The output is displayed within a minimalist grid layout to ensure clarity and ease of use.

# File Structure
app.py – Main arithmetic script.

README.md – Project documentation.

# How to Run the Application Locally
1. Make sure Python is installed.

2. Open a terminal in the project folder.

3. Run the application: python app.py

4. Input your two numbers to receive the product instantly.
